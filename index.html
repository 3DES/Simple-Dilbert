Simple Dilbert

Simple Dilbert reader for Android

  • Install fromF-Droid
  • Code on GitHub
  • View Javadoc

Simple Dilbert is application for Android, for reading Dilbert comics

Features, which might interest you:

  • Supports Android from 2.2, over 3235 devices
  • Downloading images to your device
  • Adding Widget to your launcher desktop
  • Zooming comics as needed in both display orientations
  • Selecting image quality
  • Making favorite strips
  • Caching, so you won't download single strip twice
  • All comics strips since 16.4.1989
  • Supports both large and small screens
  • Supports all device orientations
  • Published under Apache 2.0 License (Open Source)
